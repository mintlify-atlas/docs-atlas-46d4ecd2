# Built-in Themes

Iced provides 22 built-in theme variants that you can use out of the box. Each theme includes a carefully crafted color palette and extended color variations.

## Default Themes

### Light

```rust
Theme::Light
```

The default light theme with a clean, bright appearance.

- **Background:** `#FFFFFF`
- **Text:** `#000000`
- **Primary:** `#5865F2`
- **Success:** `#12664F`
- **Warning:** `#B77E33`
- **Danger:** `#C3423F`

### Dark

```rust
Theme::Dark
```

The default dark theme with a modern, low-contrast look.

- **Background:** `#2B2D31`
- **Text:** `#E6E6E6`
- **Primary:** `#5865F2`
- **Success:** `#12664F`
- **Warning:** `#FFC14E`
- **Danger:** `#C3423F`

## Using Themes

### Direct Usage

```rust
use iced::Theme;

fn theme(_state: &State) -> Theme {
    Theme::TokyoNight
}
```

### Theme Selection

```rust
const ALL_THEMES: &[Theme] = Theme::ALL;

for theme in ALL_THEMES {
    println!("{}", theme);
}
```

### Environment Variable

```bash
ICED_THEME=Dracula cargo run
ICED_THEME="Catppuccin Mocha" cargo run
```
